# StockSense AI

AI-powered stock price prediction and portfolio management system.

## Project Structure

- **data/** - Data storage directory
- **models/** - Machine learning models (Linear, Random Forest, XGBoost, LSTM)
- **sentiment/** - News scraping and sentiment analysis
- **risk/** - Risk assessment (volatility, confidence scoring)
- **portfolio/** - Portfolio simulation and management
- **app/** - Web application (Flask backend + React frontend)
